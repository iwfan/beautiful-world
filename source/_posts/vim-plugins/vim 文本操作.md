今天来说说 Vim 中跟文本操作一些相关的东西, 我们先来回顾一下 Vim 中的编辑指令:

```TXT
<number><operation><motion / text object>
```

`<number>`是指此次操作

# 常用文本操作

移动、跳转、复制、粘贴

# text object definition

aw/iw
as/is
ap/ip
ab/ib
aB/iB
and so on.
